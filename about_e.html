There are Only two e-tags and they are <em> and <embed>.

The <em> tag

The <em> element defines emphasized text. The <em> tag draws the reader's attention to the element's content, which is usually stressed in some way.

<embed>

The <embed> element defines a container for an external application or interactive content (a plug-in).
