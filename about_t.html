Semantic HTML

Below you will find examples for the "T" tags.

Definitions

<table>
Defines a table
Tag Description
<table> Represent data in a 2 dimensional grid form
<thead> Defines a set of rows defining the head of the columns
<tbody> Defines a set of rows that contain the body of the table
<tfoot> Defines a set of rows summarizing the columns
<tr> Defines a row in a table
<th> Defines a header cell in a table
<td> Defines a cell in a table
The above table incorporates the <table>, <tbody>, <td>, <th>, <thead>, <tr> elements. These are so well known I hesitate to say more.
<template>
A <template> is a container of elements that itself is not considered to be part of the DOM.
