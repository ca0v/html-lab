Semantic HTML

Below you will find examples for the "R" tags.

Definitions

<rp>
The <rp> tag is used to define what to show in browsers that do not support ruby annotations. Since all browsers support <ruby>, this tag is no longer needed.

Ruby(ルビー)
<rt>
The <rt> tag is used to define an explanation or pronunciation of characters (for East Asian typography).

Ruby(ルビー)
<ruby>
The <ruby> tag is used to define a ruby annotation. A ruby annotation is a small extra text that is displayed above or beside a character, usually used in East Asian typography.

Ruby in Japanese(ルビー)
